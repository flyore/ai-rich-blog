## AI는 질문을 잘해야 잘 써먹는다

같은 AI를 쓰더라도 어떻게 물어보느냐에 따라 결과물의 질이 완전히 달라집니다.

이걸 **프롬프트 엔지니어링**이라고 하는데, 거창한 이름이지만 핵심은 딱 하나예요.

> **"AI에게 구체적으로, 상황을 설명하며 부탁하기"**

---

## 나쁜 프롬프트 vs 좋은 프롬프트

### ❌ 나쁜 예시

```
블로그 글 써줘.
```

결과: 어떤 주제인지, 누구를 위한 글인지 모르니 AI가 대충 써줍니다.

---

### ✅ 좋은 예시

```
나는 30대 직장인이야. 퇴근 후 AI로 부업을 시작하고 싶어.
"직장인도 할 수 있는 AI 부업 5가지"라는 제목으로
블로그 글을 써줘.
- 독자: 코딩 모르는 직장인
- 톤: 친근하고 현실적으로
- 길이: 800자 내외
- 각 방법마다 예상 수익도 포함해줘
```

결과: 원하는 방향의 완성도 높은 글이 나옵니다.

---

## 좋은 프롬프트의 4가지 요소

### 1. 역할 부여 (Role)

AI에게 역할을 줍니다.

```
너는 10년 경력의 마케팅 전문가야.
```

```
너는 초등학생도 이해할 수 있게 설명해주는 선생님이야.
```

### 2. 상황 설명 (Context)

내 상황을 알려줍니다.

```
나는 강아지 용품 스마트스토어를 운영하는 초보 셀러야.
```

### 3. 원하는 것 (Task)

무엇을 원하는지 명확히 합니다.

```
상품 상세페이지 소개 문구를 3가지 버전으로 써줘.
```

### 4. 조건 (Constraints)

길이, 톤, 형식 등을 지정합니다.

```
- 200자 이내로
- 이모지 1~2개 포함
- 구매 욕구를 자극하는 톤으로
```

## 바로 써먹는 프롬프트 템플릿

### 블로그 글 작성용

```
너는 [분야] 전문 블로거야.
나는 [내 상황]이고, [주제]로 글을 쓰고 싶어.
독자는 [타겟]이야.
[조건1], [조건2]을 지켜서 써줘.
```

### 상품 설명 작성용

```
너는 쇼핑몰 카피라이터야.
아래 상품의 상세페이지 소개 문구를 써줘.
- 상품명: [상품명]
- 특징: [특징]
- 구매 타겟: [타겟]
- 분위기: [감성적/실용적/유머러스]
- 길이: [글자 수]
```

### SNS 콘텐츠용

```
인스타그램에 올릴 [주제] 관련 캡션을 써줘.
- 톤: [친근한/전문적/감성적]
- 해시태그 5개 포함
- 참여 유도 질문으로 마무리
```

## 실습: 지금 바로 해보기

아래를 ChatGPT 또는 Claude에 복사해서 써보세요.

```
너는 친절한 AI 활용 코치야.
나는 AI를 처음 배우는 완전 초보자야.
AI로 월 50만원을 버는 가장 쉬운 방법 1가지를
초등학생도 이해할 수 있게 단계별로 설명해줘.
각 단계마다 실제로 해야 할 행동을 구체적으로 알려줘.
```

결과물을 보면 프롬프트의 힘을 바로 느끼실 수 있을 거예요.

---

## 핵심 정리

1. **구체적으로** 말할수록 좋은 결과가 나온다
2. **역할 + 상황 + 요청 + 조건** 4가지를 포함하면 완벽하다
3. 마음에 안 들면 "더 자세하게", "다른 버전으로", "더 쉽게" 라고 추가로 요청하면 된다

프롬프트는 연습할수록 늘어요. 오늘부터 하루 3번만 AI에게 말을 걸어보세요. 🚀
