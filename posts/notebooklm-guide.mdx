## 노트북LM이 뭔가요?

구글이 만든 **완전 무료 AI 도구**입니다.

내가 올린 PDF, 유튜브 영상, 웹사이트 주소를 AI가 읽고 나서:

- 핵심 내용을 요약해줍니다
- "이 내용이 맞나요?" 질문에 답해줍니다
- 두 AI가 대화하는 팟캐스트를 만들어줍니다
- 자료들을 비교해서 분석해줍니다

> 💡 쉽게 말하면: **내 자료를 다 읽은 AI 비서가 생기는 것**입니다.

---

## 어디에 쓸 수 있나요?

| 상황 | 활용법 |
|------|--------|
| 긴 PDF 보고서 | 핵심만 요약해달라고 |
| 유튜브 강의 | 내용 정리 + 퀴즈 만들기 |
| 논문, 계약서 | 어려운 내용 쉽게 설명 |
| 여러 자료 비교 | 공통점·차이점 분석 |
| 블로그 글 준비 | 자료 수집 후 아이디어 뽑기 |

---

## Step 1: 노트북LM 접속하기

1. 구글 계정으로 로그인 (없으면 무료 가입)
2. 주소창에 입력: **notebooklm.google.com**
3. 파란색 **"노트북 만들기"** 버튼 클릭

---

## Step 2: 자료 올리기

접속하면 가운데에 **"소스 추가"** 버튼이 보입니다.

올릴 수 있는 자료 종류:

- **Google Docs** — 구글 문서
- **Google Slides** — 구글 프레젠테이션
- **PDF 파일** — 내 컴퓨터에서 업로드
- **텍스트 파일** — .txt 파일
- **유튜브 링크** — 영상 URL 붙여넣기
- **웹사이트 주소** — URL 입력

### 따라해보기: 유튜브 영상 올리기

1. 유튜브에서 영상 하나를 골라 주소를 복사합니다
2. 노트북LM에서 **"소스 추가"** 클릭
3. **"URL"** 탭 선택
4. 복사한 주소 붙여넣기
5. **"삽입"** 버튼 클릭

잠깐 기다리면 AI가 영상 내용을 모두 읽습니다.

---

## Step 3: AI에게 질문하기

자료가 올라가면 오른쪽에 채팅창이 생깁니다.

여기서 질문을 하면 **내가 올린 자료를 바탕으로** 답해줍니다.

### 바로 써먹는 질문 예시

```
이 내용을 3줄로 요약해줘.
```

```
이 자료에서 가장 중요한 핵심 5가지를 뽑아줘.
```

```
초등학생도 이해할 수 있게 쉽게 설명해줘.
```

```
이 내용으로 블로그 글을 쓰려고 해. 목차를 만들어줘.
```

```
이 자료에서 나오는 단어 중 어려운 용어들을 정리해줘.
```

---

## Step 4: 노트 저장하기

AI 답변 오른쪽의 **"노트에 저장"** 버튼을 누르면 중요한 내용을 따로 모아둘 수 있어요.

나중에 노트들을 모아서 블로그 글이나 유튜브 대본으로 쓸 수 있습니다.

---

## Step 5: 팟캐스트 만들기 (가장 신기한 기능!)

이 기능은 처음 쓰면 정말 놀랍습니다.

1. 자료를 올린 후 왼쪽 하단의 **"오디오 개요"** 클릭
2. **"생성"** 버튼 클릭
3. 1~2분 기다리기

그러면 **두 명의 AI가 내 자료 내용으로 대화하는 팟캐스트**가 만들어집니다!

영어로 생성되지만, 한국어 자료를 올려도 영어로 요약해서 팟캐스트를 만들어줍니다.

---

## 수익화에 활용하는 법

### 블로그 글 작성

1. 쓰고 싶은 주제의 자료 (PDF, 유튜브 등) 3~5개 올리기
2. "이 자료들을 바탕으로 블로그 글 목차 만들어줘" 질문
3. 각 섹션 내용 질문해서 초안 완성
4. ChatGPT나 Claude로 다듬기

### 유튜브 대본 만들기

1. 참고할 유튜브 영상 2~3개 올리기
2. "이 영상들의 핵심 내용을 합쳐서 5분짜리 유튜브 대본 만들어줘"
3. 완성된 대본으로 영상 촬영

### 전자책·강의 자료 제작

1. 관련 PDF 자료 여러 개 올리기
2. 챕터별로 내용 정리 요청
3. 노트에 저장해서 전자책으로 편집
